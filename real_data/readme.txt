1. The dataset is part of the US road network (in particular should be part of New York). 

2. The file xx_coor.txt contains the coordinates of each node, for example 
(0 -73.530767 41.085396) means the node with id 0 has a latitude and longitude as follows:
41°05'07.4"N 73°31'50.8"W

3. The file xx_edge.txt contains number of vertices and number of edges in the first line, 
the rest are the edges in the form of (id1, id2, distance), where the distance is measured in meters.
